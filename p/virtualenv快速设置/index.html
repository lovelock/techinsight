Virtualenv快速设置

Apr 10, 2024
1 minute read

记录一下快速配置virtualenv。

1. 安装virtualenv

macOS

1

brew install virtualenv

Linux(以Debian为例)

1

sudo apt install virtualenv

2. 指定版本

1

virtualenv venv --python=3.11

这时会自动在 venv下复制一份全新的3.11的python环境

注意：这种写法只能使用系统已经安装的Python版本，否则需要指定Python的安装路径

3. 让指定的版本生效

1

source venv/bin/activate

这时命令行提示符前会有一个 (venv)的前缀，如下，这时在这里执行 pip install就不会影响到别的环境。

1

(venv) frost@debian:~/workspace/

4. 不再使用这个Python环境

1

deactivate
